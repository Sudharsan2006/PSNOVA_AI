ERROR 1: "Root Directory" is wrong
-------------------------------------------------------------------
❌ WRONG: 
   - "server" (just the folder name alone)
   - "/server" (with forward slash)
   - "C:\Users\...\server" (full path)

✅ CORRECT:
   - Leave it EMPTY (blank) OR
   - Type: server

FIX: Clear the Root Directory field completely and leave it BLANK


ERROR 2: GitHub repo not authorized
-------------------------------------------------------------------
If you don't see your repo in the dropdown:
   1. Click "Connect account" 
   2. Click "Authorize Github"
   3. Select Sudharsan2006/PSNOVA_AI
   4. Make sure "master" branch is selected


ERROR 3: Build or Start command is invalid
-------------------------------------------------------------------
✅ CORRECT values:
   Build Command: npm install
   Start Command: npm start

❌ WRONG values:
   - npm install . (don't add the dot)
   - npm run start (wrong!)
   - start (incomplete!)


ERROR 4: Required fields missing
-------------------------------------------------------------------
These MUST be filled in:
   ✅ Name: team-ai-backend
   ✅ GitHub repo: selected and connected
   ✅ Build Command: npm install
   ✅ Start Command: npm start
   ✅ Environment Variables: Add at least MONGO_URI and NODE_ENV


ERROR 5: Plan not selected
-------------------------------------------------------------------
Make sure you select either:
   - Free (takes longer, but free)
   - Starter ($7/month, faster deploys)
